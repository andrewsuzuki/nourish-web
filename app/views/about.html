About

Hi! My name is Andrew Suzuki. I'm a computer science major here at UConn (2018).

Nourish is a modern set of apps I wrote in the Summer of 2015 as a replacement for UConn's lacking dining hall menu pages. Nourish currently provides:

  • Clean and intuitive interfaces for finding menu information
  • A mobile app for iOS and Android, which allows one to save a list of their favorite foods and be alerted when the foods are available anywhere
  • A web app that looks great on any device, automatically scaling for desktops, laptops, tablets, and phones
  • More coming soon...

Development

Nourish is open source software. If you're a developer, feel free to contribute on GitHub. Nourish is engineered with full-stack Javascript. Node/Express handles the backend API, AngularJS orchestrates things on the front-end, and Ionic/Cordova provides the base for the cross-platform mobile app.

There are three repositories:

  • nourish-server — exposes a RESTful web API and periodically scrapes UConn's menu pages
  • nourish-web — this web app (hosted through GitHub Pages - see gh-pages branch)
  • nourish-app — the mobile app

To work on any of these, you need to download Node.js (V8 Javascript platform), NPM (Node Package Manager), and Gulp (build system). You might also need Bower (front-end dependency manager), Ionic (mobile app framework), Cordova (mobile app platform), or whatever else it warns you about not having.

If you're not a developer, but have an idea for a feature, feel free to contact me on Facebook, or through my prototypical UConn email address.
